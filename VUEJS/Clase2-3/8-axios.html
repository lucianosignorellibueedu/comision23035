Axios para realizar una solicitud GET a la API JSONPlaceholder, que proporciona datos de usuarios. Cuando se monta el componente Vue (en el ciclo de vida mounted), llamamos al método fetchUsers. Dentro de fetchUsers, usamos Axios para hacer la solicitud GET a la URL. En el método .then, asignamos los datos de respuesta a la propiedad users del componente. En caso de error, capturamos la excepción y la mostramos en la consola. En la HTML, utilizamos v-for para iterar sobre la matriz users y mostrar los nombres de los usuarios en una lista.
